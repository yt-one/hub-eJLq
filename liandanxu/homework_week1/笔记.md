## 了解使用sklearn库进行机器学习
- 机器学习的步骤：数据预处理、特征工程、模型训练、模型评估
- 机器学习任务分类：分类、回归、聚类、降维等
- 常用模型在sklearn库的调用方式：
- 分类：LogisticRegression,DecisionTreeClassifier，KNeighborsClassifier
- 回归：LinearRegression,DecisionTreeRegressor
- 聚类：KMeans
- 降维：PCA
- sklearn库常用接口：
- 模型创建：model = Model() # 根据实际情况选择不同的模型
- 模型训练：model.fit(X,y) # X 训练集 y 真实值
- 模型预测：model.predict(X) # X 测试集
- 模型评估：model.score(X,y) # X 测试集 y 真实值

## 通过百炼调用qwen
- 可通过openAI的标准协议，获取到自己的api-key后，进行登录
```python
from openai import OpenAI
client = OpenAI(
    api_key="sk-xxx", # 账号绑定的
    # 大模型厂商的地址
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
)
```
- 登陆后，可通过如下方式进行单词简单问答：
```python
completion = client.chat.completions.create(
    model="qwen-plus", # 模型的代号
    messages=[
        {"role": "system", "content": "You are a helpful assistant."}, # 给大模型的提示词
        {"role": "user", "content": "你是谁？"}  # 用户的提问
    ]
)
print(completion.choices[0].message.content)
```
